---
description: Run the Chat Backend and Flutter App
---

1. Stop the existing Node.js server if running.
2. Navigate to the `chatdb` directory.
   ```bash
   cd chatdb
   ```
3. Install dependencies (if needed, mainly `axios` which was added to package.json but might need install if not present).
   ```bash
   npm install
   ```
4. Start the Node.js server.
   ```bash
   node server.js
   ```
5. Run the Flutter app.
   ```bash
   flutter run
   ```
